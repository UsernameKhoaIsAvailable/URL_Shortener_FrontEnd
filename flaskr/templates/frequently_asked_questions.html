T.LY URL Shortener FAQ

Are short links case sensitive?

Yes, short links are case sensitve.

How do I view stats for a short URL?

To view stats for any URL, just a + sign to the end. Here is an example t.ly/P9IK+

Can I password protect a link?

Yes, a password can be added to any short url and only those with it will be able to view the link. Here is a guide on how to password protect a short url.

How do I report a short url that redirects to spam or a malicious site?

Please send malicious short urls to support@t.ly

Can I Import Short URLs?

Yes, you can import short URLs into T.LY. Create an account to Get Started.

Does T.LY integrate with Zapier?

Yes, you can easily create short URLs using Zapier. Check out the integration here.

Does T.LY support bulk shortening of links?

Yes, we allow users to upload CSV files to bulk shorten URLs. To get started, please register an account.

Can I set a link to expire?

Yes, a expire date or number of views can be added to any short url. After the date/view count, the link will no longer work.

What are Smart URLs?

Smart URLs redirect users to different destinations based on their location, device, or browser. This allows you to share one URL to target multiple destinations. Here is an example that will redirect users based on their device to the Link Shortener extension install page depending on their browser. Give it a try: https://t.ly/extension

Can I create free short URLs?

Yes, T.LY allows users to create short urls from the homepage and extension.

Do we offer Yearly Plans?

Yes, we do have yearly plans that offer one month free. Check them out on the manage subscription page.

Can I add my own custom domain?

Yes T.LY is a url shortener that supports custom domains. Adding your own domain is a great way to brand urls. You must create an account to add your own domain.

How do I fix DNS_PROBE_FINISHED_BAD_CONFIG Error?

If you are experiencing this error, we recommend you update your local DNS records to either Cloudflare or Google's DNS. Please refer to this guide for additional information.

Do you have a browser extension?

Yes, we have a browser extension that works for Chrome, Firefox, Edge, and Opera. Generate an API Key then add it to the extension under options. If you need help please see this guide.

What happens to my links when my trial or plan expires?

Your links will be in a frozen state until your account subscription is reactivated. You will not be able to manage your links or browse to them until you have an active subscription.

How do I edit a short links?

To edit short links, you have to create an account. From the link dashboard you can update any short link. You can change the URL ending, destination URL, add a password, and auto expire a short link.

How the redirection is made?

T.LY has a database that associates the long url with a short url. When a user tries to access the short url, the browser will instantly redirect to the original page (long url).

How can I Expand a Short link?

Link Unshortener supports all link shortening services and will programmatically follow url redirects to their final destination. Enter a short link to will discover the final destination. Use Link Unshortener to review short links before browsing to them.

Does T.LY support Two Factor Authenication(2FA)?

T.LY places a strong emphasis on security and supports Two-Factor Authentication (2FA), enhancing the security of user accounts. This feature adds an extra layer of protection by requiring a second form of verification, such as a code from an app, in addition to the standard username and password. The option of 2FA helps prevent unauthorized access, ensuring that even if a password is compromised, the account remains secure. Two-Factor Authentication (2FA) demonstrates our commitment to user security and trust, making T.LY a reliable choice for managing and sharing shortened URLs.

How long will my T.LY URL be available?

Our URL shortening service at T.LY offers reliable and secure short links that do not expire as long as they are created and maintained per our Terms of Use. If your link is created under an account, a valid subscription is required to remain active.

Will URL Shorteners hurt my site's SEO?

No, using T.LY, does not hurt your SEO. Google and other search engines will pass pagerank value to the final url rather than the short link. 301 redirect is a permanent redirect from one URL to another. 301 redirects send site visitors and search engines to a different URL than the one they originally typed into their browser or selected from a search engine results page.

My Pixel Tracking is Not Working

Ad-blockers or browser extensions can interfere with pixel tracking. These tools are designed to block advertisements and scripts from loading on web pages, including tracking pixels. When a user has an ad-blocker or similar extension enabled, it can prevent the pixel code from being executed, resulting in inaccurate tracking data. Ad-blockers and browser extensions are widely used, with a significant portion of internet users relying on them to enhance their browsing experience and protect their privacy. As marketers, it's important to acknowledge the impact of these tools on pixel tracking and respect the privacy of users.

Does Google still have a URL shortener?

No, Google discontinued its URL shortener service in 2019. The URL shortener was used by many people to shorten long links, and this made it easier to share links on social media and other platforms. Google announced on March 30, 2018, the discontinuation of the service on March 30, 2019.
